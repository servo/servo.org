This Week In Servo 13

What's up with Servo for the week of 18 - 25 Nov 2014

Posted 2014-11-25

This week, we merged 26 pull requests

Next week we’ll be in Portland, and holding “getting started” sessions after hours for interested people. More information here. Do come; there are stickers to be had!

Notable additions

  • Glennw integrated glutin for linux. Glutin is a pure rust alternative to GLFW, you should check it out!
  • Patrick made some more incremental reflow improvements
  • Mike filled in the string multimap bits of CEF and made some other embedding changes
  • Ms2ger landed a web platform test upgrade
  • Kshitij fixed a panic in SnifferManager
  • Donald fixed the displaying of form buttons
  • Rohan implemented document.createAttribute()
  • Matt Rasmus made a bunch of mach improvements, including a mach update and the ability to run with a debugger

New contributors

8 new contributors this week, hooray!
