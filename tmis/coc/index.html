Code of Conduct

The Servo Project follows the Rust Code of Conduct, with two differences.

Our code of conduct applies to the Servo Zulip channels, GitHub repositories, and all official Servo venues, rather than the Rust project.

If you are subject to or witness unacceptable behavior, or have any other concerns, please notify one of our designated contacts:
