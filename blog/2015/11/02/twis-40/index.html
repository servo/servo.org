2015-11-02 What's up with Servo for the week of 27 Oct 2015 - 02 Nov 2015

In the last week, we landed 97 PRs in the Servo organization’s repositories.

Notable additions

  • Emily Dunham and jdm did some work getting http://servo.org/starters working!
  • Manish added support for @bors-servo: try for people who don’t have full reviewer priviledges in Servo
  • Bobby Holley added a bunch of optimizations to how we handle restyling
  • nxnfufunezn added support for brotli http responses
  • Till cleaned up our script loading and execution
  • David Zbarsky added more calc expressions in CSS

New Contributors

  • Axel Solis Trompler
  • Nova Fallen

Meetings

At last week’s meeting, we discussed WebRender, automated testing, ipc-channel fallthroughs, and DOM invalidation changes.
