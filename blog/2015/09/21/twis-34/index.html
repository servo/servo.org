These Weeks In Servo 34

2015-09-21 What's up with Servo for the weeks of 28 May 2015 - 10 Oct 2015

It’s been a long time! In the 16 weeks since the last “This Week in Servo,” we have:

  • Moved from Critic to Reviewable for our code reviews

  • Welcomed Ms2ger as a full-time member of the Servo team, working remotely from Belgium

  • Welcomed Anthony Ramine and Corey Farwell as reviewers.

  • Upgraded to a modern version of our JavaScript engine, Spidermonkey via some amazing work by Michael Wu

  • Attended an all-Mozilla workweek in Whistler, BC, with Servo training, and meetings on graphics, networking, FFOS, houdini, and houdini again

  • Landed 1032 non-merge commits (around 65 per week)!

  • A Mozilla+Samsung presentation on Servo at LinuxCon Japan

  • A blog post by Nicholas Nethercote about our profiling code

  • Patrick added interprocess communication via ipc-channel

  • Sean landed support for HTTP connection pooling

  • Sam made our networking code testable and added HSTS support

  • Michael added a test harness to run Dromaeo on Servo painlessly and automatically extract results

  • We have been tweeting more information via ServoDev, complete with an awesome but totally unofficial logo

Progress is amazing, and we apologize for the hiatus in this post!
