Made With Servo

This page showcases projects that are built with Servo or Servo components.

Browser UIs

servoshell

First-party browser UI built with egui

Verso

A browser UI for Servo that implements the UI as HTML using Servo's own web rendering support

Servo Qt

A Servo Webview as Qt widget, enabling browser UI built in Qt

Moto

A fork/extension of servoshell that aims to add features such as bookmarks, history and download management.

Cuervo

A text-based UI for Servo

Wry Backend

A Servo backend for Wry, the webview library used by Tauri
