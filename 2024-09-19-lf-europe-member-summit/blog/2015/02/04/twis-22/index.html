This Week In Servo 22

2015-02-04 What's up with Servo for the week of 28 Jan - 3 Feb 2015

This week, we merged 66 pull requests

It’s Servo’s third Github-birthday! The repository was created on 5 February, 2012, though the first commit was three days after that.

No birthday is complete without food. Servo got cookie support today!

Josh also gave a talk on contributing to Servo at FOSDEM. The video isn’t up yet, but you can find the slides here.

Notable additions

  • Josh and Shamir added cookie support to Servo. Now you can log in to Github and some other sites. Reddit doesn’t work yet, however.
  • Manish added support for SSL certificate validation via OpenSSL. Your certificate is bad, and you should feel bad
  • We landed a Rust upgrade!
  • Yodalee added form submission support for buttons
  • Ms2ger switched some of our DOM pointers to use NonZero<T>
  • Arpad added compile-time lowercasing as a syntax extension and switched our DOM macros to use it
  • Gilles refactored event code to avoid mixing up the trustedness
  • Glenn removed our GLFW version. We now use glutin on both Linux and OS X.
